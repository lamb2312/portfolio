# ポートフォリオサイト制作アイデア

## 仕事絵（Works / Commercial）と趣味絵（Personal / Original）の分け方

* **仕事絵（Works / Commercial）**
  * **目的**: 企業や法人のクライアントに向けた実績アピール。「ビジネスとしてどのような形式で納品を行ったか」を示す。
  * **載せるべき情報**: クライアント名、プロジェクト概要、担当範囲（キャラデザのみ、背景込み等）、制作環境・ツール、制作期間など。

* **個人依頼・コミッション（Skeb, Pixivリクエスト等）の扱い**
  * **基本方針**: 基本的には「Works（仕事絵）」に入れて問題ない。（「お金を頂戴して期日・要望通りに納品した実績」として立派なアピールになるため）
  * **分け方の工夫**: もし法人案件（Commercial）と明確に分けたい場合は、Worksの中に「Commercial（企業案件）」と「Commission / Request（個人依頼）」という2つのカテゴリー（タグ）を作ると、見る側にも親切。
  * **注意点**: クライアント（依頼者）からポートフォリオ掲載の許可が下りている、もしくは規約上掲載が許可されているもののみを載せること。

* **趣味絵（Personal / Original）**
  * **目的**: ファンに向けた世界観の提示や、将来やりたい方向性のアピール。
  * **載せる設定**: 自分の持ち味、一番得意な表現。「こういう世界観の絵も描けます」という未来の仕事へ繋がるカタログ的役割。

## おすすめのサイト構成（ページ割り）

シンプルで迷わない構成が好まれる。

1. **Home（Top）**: 一番自信のある絵や最新作を大きく配置し、インパクトを出す。
2. **Gallery / Portfolio**: 「Works」と「Personal」のタブやボタンで切り替えられるようにする。
3. **About / Profile**: 
   * ペンネーム、簡単な自己紹介、活動実績
   * 得意なジャンル（萌え系、厚塗り、線画メインなど）
   * 使用ツール（Clip Studio Paint, Photoshopなど）
   * インボイス対応の有無（企業向けにあると好印象）
4. **Links（SNS・外部サービス）**:
   * **X (Twitter)**: 最新の活動状況や人柄がわかるため、クライアントもファンもよく見る。なるべく「絵のお仕事専用」に近いアカウントが望ましい。
   * **Pixiv**: 作品の全体像（過去作含む）を見たいファンや、ポートフォリオに入りきらなかった作品の保管庫として機能する。
   * **FANBOX / Fantiaなど**: コアなファン向けの窓口。クライアント向けではないが、クリエイターとしての活動規模をアピールする要素になる。
   * **Skeb / Pixivリクエスト**: 個人からの依頼を直接受け付ける「入り口」。
     * **両立は全く問題なし**: どちらも利用層に違いがあるため、両方のリンクを並べて設置しても違和感はない。
     * **メリットの使い分け**:
       * **Skeb**: 「とにかくラフに、やり取りなしで描いてほしい」という投げ銭感覚のファンや海外ユーザーに強い。
       * **Pixivリクエスト**: Pixiv経由でファンになった人（国内層）が依頼しやすく、「相乗り」機能で金額が上乗せされる面白さがある。
     * **注意点**: どちらも開けておく場合は、自分自身のキャパシティ管理（スケジュール）だけ気をつけておくこと。
5. **Contact**: 
   * お仕事の依頼フォームやメールアドレス（企業・法人窓口）。
   * 「現在〇〇月以降の着手で受付中」「ご予算の目安」などを併記すると親切。

## デザインや運用のポイント

* **主役はあくまで「絵」**: サイト自体のデザインはシンプル（白、黒、グレーなどの無彩色ベース）にし、絵の印象を邪魔しないようにする。
* **画像保護の意識**: 無断転載やAI学習対策として、ウォーターマーク（透かし）を入れたり、右クリック保存禁止などの工夫を検討する。
* **スマホ対応（レスポンシブ）は必須**: 企業の担当者も最初はスマホからチェックすることが多い。

## 掲載する絵の選び方（作品の絞り込み）

* **「最新のベストな作品」を優先する（一番重要！）**
  * ポートフォリオは「今、何が描けるか」「今後どういう仕事がしたいか」をアピールする場なので、昔の作品をすべて載せる必要はない。
  * 特に**現在のスキルレベルと比較して「荒削りだ」と感じる過去の作品（例：昔挑戦したグリザイユなど）は、載せない勇気を持つ**ことも大切。
  * 全体の評価は「載せている中で一番クオリティの低い作品」に引っ張られる傾向があるため、直近の自信作を中心に**10〜20点程度**に厳選するのが圧倒的におすすめ。
* **仕事絵（依頼で描いた絵）にも同じ基準を適用する**
  * 「仕事で描いたから全部載せなきゃ」→ **これは不要**。個人サイトのポートフォリオは「実績の網羅的な履歴書」ではなく「今の最高水準を示すベスト盤」。
  * 3年前など過去の仕事絵で、今と比べて完成度が劣ると感じるものは**思い切って外してOK**。
  * 「実績の数が多い」ことよりも「載っている全作品のクオリティが高い（＝最低ラインが高い）」方が、クライアントに圧倒的に信頼される。
  * ※仕事量の証明が必要になるのは、エージェンシー登録や就活用など別の場面。個人のWebポートフォリオでは不要。
* **特定の画法やテーマは「今のスキルで描いたシリーズ」としてまとめる**
  * 現在の画力で描かれた別テイストの絵があれば、シリーズとして見せることで「表現の引き出しの多さ」をアピールできる。
  * ただし、「色々な描き方ができる（バリエーションの多さ）」ことよりも「ひとつひとつのクオリティが高い」ことの方が圧倒的に優先される。
* **「今後やりたい仕事」に近い絵を目立たせる**
  * これから増やしていきたいタッチや路線の絵を一番目立つ場所（トップページなど）に配置する。

## サイト運用の基本方針

* **「直近の旬な絵（最新の自信作）」を中心に小さく始める**
  * 最初から過去の作品を大量に引っ張り出して完璧な網羅を目指す必要はない。
  * 現在の全力が伝わる数枚だけで、まずはWeb上に自分の「名刺・カタログ」を公開してしまうのがおすすめ。
* **質の高い新作ができるたびに「入れ替えて」徐々に更新する**
  * 新作ができたらサイトに追加し、相対的にクオリティが見劣りするようになった古い作品を下ろしていく。
  * この「新陳代謝」を続けることで、常に一番クオリティが高く、今の状態を反映した最強のポートフォリオを維持できる。

## 画像フォーマットについて（軽量化戦略）

* **「WebP（ウェッピー）」の積極的な採用を推奨**
  * 高速なサイト表示は作品の印象アップに直結するため、サイズの大きいPNGやJPEGよりもWebPを活用するのが非常に効果的。
  * **メリット1**: 画質をほとんど落とさずにファイルサイズを25〜30%ほど劇的に軽量化できる。（特にスマホからの閲覧時に威力を発揮）
  * **メリット2**: Googleなど検索エンジンの評価（SEO）でも表示速度が重視されるため有利。
  * **メリット3**: 現在はあらゆるモダンブラウザ（Safari、Chrome、Edgeなど）で標準対応しているため、表示されない懸念も解消されている。

## 集客・告知の戦略（Xを中心とした発信）

**目的**: Skeb・Pixivリクエストでの依頼を増やすこと。（現状：Skeb実績なし、Pixivリクエスト納品2件）

### サイト公開時にやること
* 一番自信のある絵を1枚添付して「ポートフォリオサイトを作りました！」と告知ツイートを投稿する。
* サイトURL + Skeb + Pixivリクエストのリンクをまとめて載せておく。
* このツイートを**固定ツイート**にして、プロフィール訪問者の目に必ず入るようにする。

### 普段の発信で意識すること
* 完成イラストを投稿するたびに、「**Skeb・Pixivリクエスト受付中です！**」と毎回さりげなく一行添える。（認知の第一歩は「この人は依頼を受けてくれる人だ」と知ってもらうこと）
* 納品した絵（公開可能なもの）は「Pixivリクエストで描かせていただきました！」と積極的にポストする。実績が少ない今こそ、依頼実例を見せることが信頼感に直結する。
* 描きたいテーマ・ジャンルの絵をコンスタントに投稿し続ける。「この人にこういうジャンルの絵を頼みたい」と思わせることが依頼のきっかけになる。

### リンクの貼り方（本文 vs セルフリプライ）
* **外部リンクは本文に入れず、セルフリプライにぶら下げるのが鉄則**
  * Xのアルゴリズムは「ユーザーをX内に留めたい」設計のため、外部リンク付きツイートはインプレッション（表示回数）が下がる傾向がある。
  * 本文にリンクを入れると、せっかくの良い絵でも人の目に届きにくくなるリスクがある。
* **「受付中です！」の文言は本文に入れてOK（リンクではないため影響なし）**
  * 趣味絵・仕事絵問わず、毎回の投稿で「Skeb・Pixivリクエスト受付中です！」と一行添える。
  * これにより「この人は依頼を受けてくれるんだ」という認知が自然に広がる。
* **おすすめの投稿テンプレート（趣味絵・仕事絵共通）**:
  * **メインツイート（本文）**: 絵の画像 + キャプション + 「Skeb・Pixivリクエスト受付中です✨」（リンクは入れない）
  * **セルフリプライ（ぶら下げ）**: 「📎ポートフォリオ → URL」「📎Skeb → URL」「📎Pixivリクエスト → URL」をまとめて記載。
  * → メインツイートが最大限拡散され、興味を持った人がリプ欄でリンクにたどり着ける。
* **Xのプロフィール欄にポートフォリオURLを設定しておく**
  * 毎回「実績はプロフィールのサイトから！」と書かなくても、プロフィールを見た人が自然にサイトへたどり着ける。
* **例外：固定ツイートは本文にリンクを入れてOK**
  * 固定ツイートはタイムラインの拡散ではなく、プロフィール訪問者が見るものなので、リンクを直接本文に記載して問題ない。

### 実績が少ない時期の施策
* ハッシュタグの活用: `#イラスト好きな人と繋がりたい`、`#Skeb募集中`、`#コミッション` など。
* 「フォロワーさんのオリキャラ描きます企画」など、認知拡大のための無料企画を打ち出してみるのも有効。（拡散されやすく、ファン獲得のきっかけになる）
